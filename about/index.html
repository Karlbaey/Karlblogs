1 欢迎你！

欢迎你来到卡尔白的博客站！

如你所见，karlbaey.top（下称本站）是卡尔白（en_US = Karlbaey），也就是我的个人博客网站。本站脱胎于真理博客版，随后慢慢来到了全球最大同性交友程序员交流网站 GitHub 上。我个人暂时没有写代码的需求，所以本站就来当作我记录生活的地方吧！

若不做特殊说明，“我”“我本人”“卡尔”均指卡尔白。

3 常见问题解答

Q3.1 我读完了，现在我能干什么？

A：

太多了。

你能读读我的博客，要是发现什么有趣的也可以联系我。

好像也不怎么多。

Q3.2 卡尔，你写的挺不错的，我能给我的朋友家人们看看吗？

A：

当然可以，只要你遵照2 关于本站的全部内容就可以了。不过记得，不要因为观点不同就吵架。

Q3.3 你的博客有友站吗？

A：

有的兄弟，有的，请点这里。

Q3.4 我该怎样联系站长？

A：

分三种途径：

  1. 针对博客文字本身的疑问或见解，直接在对应的博客页发评论。评论服务由 giscus 提供。
  2. 如果对站点的 HTML 或是 JavaScript 代码等有建议，请到这里发 issue。在发 issue 前我非常建议你先读读其他朋友们发的 issue 并参考标签，这或许对你表达建议有些帮助。
  3. 想找卡尔聊天的可以直接加我的 QQ 2454161978 或者与我互发邮件 karlbaey101@outlook.com。

除此之外，我也建议你了解以下这些常用的电子邮箱账号：

  • info@karlbaey.top ——本站的新闻机器人（正式名称：Karlbot-info ），会不时给订阅者发邮件通知网站的重大事项，兼任发布与友站的官方互动消息。活动通告也由这个邮箱发出。需要注意的是，这个邮箱不具有收信功能，如果你想反映问题，请往下看。

    • 不过虽然说是机器人可邮件都是我亲手写的😡

  • inbox@karlbaey.top ——本站专用收信箱（正式名称：Karlbox-in ），与发 issue 具有同等效力，我个人更推荐前者。与 Karlbot-info 相反，这个邮箱没有发信功能，因此送至这个信箱的邮件很可能没有回音，这也是我推荐发 issue 的理由。

  • 待更新……
