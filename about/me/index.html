关于我

你好，我是@卡尔白！也就是这个网站的站长！本文有一个更正式的版本，请看这里

我吗？其实还是有很多想说的吧。

名字的由来

Karlbaey *（跟我一起读：/‘kɑ:rlbɑ:ɪ/）*中文音译为卡尔白，分成两部分：

  1. *（上半）*Karl：来自卡尔·马克思的名字。
  2. *（下半）*baey：ey来自于酶的英文 Enzyme 的第一和第四位字母，ba 在其中充当连接作用，并不是必要成分。

说是这么说拼写我名字的时候也不要拼成 Karley 了:) （话说 Karley 很像一个女生的名字，应该可以设定成我的妹妹之类的……）

开端大概在 2022 年末，当时的我还用着相当逆天的网名*（CthulhuXD）*，后来读了一些书发现它实在不合我的口味，就产生了换昵称的想法。我是属于那种喜欢在全部社交平台都用同一个昵称的人，就很谨慎地用拉丁字母排列组合出了现在你看到的这个昵称。

不知不觉又是一个三年……

说实话，我很喜欢这个名字，这也直接导致了我选择域名时优先考虑了 Karlbaey.top。

我是谁？

没有想象中的那样重要啦。

我更像互联网上的一团灵魂，到处都有我的足迹，但找不到我存在的证据。就连你正在看的这个页面，也是一样。它可以是一个人类在背后操控，或者是一个人工智能在管理，甚至可以是 GitHub Actions 定时发布的产物。看到了吧？连我亲自去推送都不必要。这个网站的一切，全在于你是否相信屏幕对面的是否是一个人。

但我觉得很多人都会相信，是屏幕对面的一个人写下了这段文字。这是互联网的幸运：它是由无数个人连接起来的，自然只有在屏幕对面是个人时，人们才会静下心来认真读读这段文字。至于 TA 具体是谁，那是次要的东西，TA 可以是一个苦打工久矣的社畜，可以是一个靠互联网逃离现实的高中生，甚至可以是一个刚满十六岁的白毛赤瞳巨乳精灵，是什么都可以，只要 TA 不是一行行代码堆砌的，是充满人情味的，是一个在地球上生活的人，这就够了。

仔细一想，这正是互联网迷人的地方。所以关于我的问题，随你喜欢就好，不必上纲上线。

为什么要搭网站？

给互联网上的自己一个容身处吧，叫避难所也可以。

现实里活得太辛苦了，给自己一个容身之处续续命，我觉得很有必要，所以平常抠抠搜搜而且懒的我才会下决心买下域名，自己零基础搭起了这个网站。因为代码技术太薄弱，中间遇到的难处也相当多，不过都过去了，现在的我只需要享受过去的自己留下的成果就好。

这个网站相比其他的大博客站来说显得确实寒酸，不过我不嫌弃，这就够了。

好了，说了很多有的没的，还是扯点有用的吧。

高中选科是物化生，因为文科太费力。不过出人意料的是我英语出奇的好，以至于很多次考试都是因为英语才避免了挂科的命运。数学……还是算了吧，这就留给喜欢的人，我不喜欢数学。

我没什么能拿得出手的绝活，就乐意摆弄摆弄电脑。这个网站前后搞了两次，因为用不上 Wordpress 很是吃了点苦头，不过慢慢好起来了，它也安稳地度过了满月，我希望能看到这个小站周岁的那一天。

Python 小懂，我做批量处理始终是 Python 一家独大。其他嘛……几乎可以说一窍不通，譬如主流的 C 家族、Java、JavaScript 跟编程语言都一窍不通。不过是有例外的，我很喜欢 Markdown，顺带着把 HTML 语法也学了，在编写博客的时候省了我很大的事。

排版癖好吗？喜欢在段与段间空一行，中文和拉丁字母或是阿拉伯数字中间喜欢留一个空格（也就是为人津津乐道的“盘古之白”）。

我的故事从此分开两条，一条从原子的世界里淌过，或许哪天遇到一座高山就从此止步不前；另一条留在无线电波里，幸运的话，它能飞离我的大脑，飞向太空，飞向没人去的地方，被一个永远不可能见面的人看到，然后被扔进垃圾堆。

我不在乎，因为我真的活过，即使是一错再错的纠结，也会变成无名者的挽歌。

Not the end
